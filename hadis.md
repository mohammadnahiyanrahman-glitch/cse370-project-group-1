## How to setup the database
> 1. Start xampp
> 2. Start Apache & mysql
> 3. Click mysql Admin
> 4. import setup.sql


## How to Test and Run
> [!IMPORTANT]  
> 1. Start your XAMPP Apache and MySQL services.  
> 2. paste `polititrack` folder at `C:\xampp\htdocs\polititrack`.
> 3. Go to `http://localhost/polititrack` in your browser.
